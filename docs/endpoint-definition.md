
# Endpoint Definitions

This document describes the efficacy endpoints used in the
oncology Objective Response Rate (ORR) table.

---

## Best Overall Response (BOR)

Best Overall Response is derived from tumor response assessments
and categorized as:

- CR – Complete Response
- PR – Partial Response
- SD – Stable Disease
- PD – Progressive Disease

Only records with `PARAMCD = "BOR"` are considered.

---

## Objective Response Rate (ORR)

Objective Response Rate is defined as the proportion of subjects
with a Best Overall Response of:

- Complete Response (CR), or
- Partial Response (PR)

ORR is summarized by treatment group.
